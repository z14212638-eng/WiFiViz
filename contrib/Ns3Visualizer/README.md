# Ns3Visualizer contrib module

This directory is the ns-3 contrib module for `ns3-Visualizer`. It contains the
ns-3 tracing components, helper APIs, Qt frontend, script generator, examples,
and default GUI scenarios.

For complete installation, build, running, UI, and screenshot/GIF instructions,
see the repository-level documentation:

- [English README](../../README.md)
- [中文 README](../../README.zh-CN.md)

Expected location after installation:

```text
ns-3.46/contrib/Ns3Visualizer
```
